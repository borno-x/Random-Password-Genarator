# Random Password Generator

**Author:** Borno Modak (borno-x)  
**Created:** 18-Jan-2026  
**Last Modified:** 18-Jan-2026  

A simple Python program to generate strong, random passwords with letters, numbers, and symbols.

---

## Features

- Generate passwords of customizable length (default: 12 characters)
- Includes uppercase, lowercase, digits, and special characters
- Easy to use and modify

---

## Requirements

- Python 3.x
